# qtile-config

My config for the Qtile Window Manager

### This contains configs for

- wallpapers
- alacritty
- picom
- qtile
- rofi
- dunst
- autostart script
- neovim config

#### Sync config

_sync_config.sh_

To run:

`./sync_config.sh`

#### Restore config

_restore_config.sh_

To run:

`./restore_config.sh`
